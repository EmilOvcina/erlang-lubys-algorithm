The module provides functions for creating graphs that represent common types of overlay topologies.

Authors: peressotti@imada.sdu.dk.

Description

The module provides functions for creating graphs that represent common types of overlay topologies. Graphs are directed and represented using adjacency lists.

Data Types

file_error()

file_error() = file:posix() | badarg | terminated | system_limit

graph()

graph(V) = [{V, [V]}]

A directed graph given as adjacency map. The adjacency map is a list of pairs where the first component represents the vertex (e.g. an integer) and the second component is the list of vertices in its neighbourhood.
