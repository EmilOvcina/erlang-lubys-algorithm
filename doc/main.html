Description

The module runs Luby's algorithm for Maximal Independent Set on a Graph. This is used in together with simulator and simulator_util modules.

main/3

main(X, Neighbours::[X], Nodes::[{X, pid()}]) -> none() | ok

Neighbours: is list of ids of vertices, which corresponds to the neighbours of the vertex.
Nodes: is a list of tuples mapping vertex ids to pids.

Main function running Luby's algorithm. This prints the id if the node is selected for the MIS. Should be used with the simulator module. The simulator:start function should be provided an UNDIRECTED Graph since this algorithm assumes that all graphs are undirected.
